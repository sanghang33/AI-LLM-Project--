⚾ AI 기반 KBO 타자 성적 예측 프로젝트

딥러닝과 LLM을 활용하여 KBO 리그 타자의 다음 시즌 성적 증감(Δ) 을 예측하고,
그 예측 결과에 대한 이유를 자연어로 설명하는 웹 기반 분석 시스템입니다.

📌 프로젝트 개요

최근 KBO 리그는 1,200만 관중을 돌파하며 높은 인기를 기록하고 있으며,
본 프로젝트는 단순 기록 조회를 넘어 데이터 기반으로 선수의 미래 성적 변화를 예측하는 것을 목표로 기획되었습니다.

KBO 타자 데이터를 기반으로 다음 시즌 성적 변화 예측

예측 결과를 수치 + 시각화 + 자연어 해설 형태로 제공

Streamlit 기반 웹 서비스로 구현

🎯 프로젝트 목표

KBO 타자의 다음 시즌 성적 증감(Δ = next − current) 예측

최근 성적 흐름 + 연령 효과(에이징 커브) 반영

LLM을 활용해 *“왜 이런 예측이 나왔는지”*를 설명하는 기능 구현

📊 데이터셋 구성

출처: Kaggle – KBO player dataset

원천 데이터 기간: 1982 ~ 2025

모델 학습 구간: 2015년 이후 시즌

데이터 선정 기준

2000년대 이후 데뷔한 KBO 타자

시즌 223 타석 이상 (규정 타석의 절반)

단위: 선수 × 시즌

주요 입력 변수

나이, 타석 수(PA)

타율, 출루율, 장타율, OPS

홈런, wRC+, WAR 등

특징 엔지니어링

실제 에이징 커브의 비선형성을 반영하기 위해 Age² 변수 추가

최근 성적 흐름과 출장 안정성을 함께 고려

🤖 모델 구조
예측 모델

MLP 기반 딥러닝 회귀 모델

입력: 시즌별 타자 성적 지표 + 파생 변수

출력: 다음 시즌 성적 증감(Δ)

학습 설정

Epoch: 120

타석 수 증가 시 MAE / RMSE 감소 경향 확인

과도한 상위 타자 편향을 방지하기 위해 223 타석 기준 적용

🧠 LLM 연계 (설명 생성)

로컬 LLM: Qwen

Ollama API를 통해 호출

예측 결과를 기반으로 다음 내용을 자연어로 설명:

최근 성적 흐름
